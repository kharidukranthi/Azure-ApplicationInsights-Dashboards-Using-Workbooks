# Detailed Performance & Dependency Analysis Workbook

### Overview
A deep-dive engineering dashboard that helps identify:
- Slow operations  
- Backend dependency delays  
- Query spikes  
- Throttling patterns  
- User-based analysis  
- Request → Dependency correlation  
- p50 / p95 / p99 latency insights  

### Included Modules
- Operation heatmaps  
- Dependency latency breakdown  
- Failure pattern correlation  
- Percentile trend charts  
- Investigative root cause views  
- Per-operation drilldowns

### Deployment
Same steps as Executive Dashboard:
1. Azure → Application Insights → Workbooks → New → Advanced Editor
2. Paste JSON
3. Update `{YOUR-...}` fields
4. Save
